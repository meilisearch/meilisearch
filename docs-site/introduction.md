# Introduction

This is a Sourcey-generated docs site for [Meilisearch](https://www.meilisearch.com/), an ultra-fast, open-source search engine written in Rust.

This site is being published as a candidate deliverable for Frantic bounty #46 (Publish Sourcey docs for a second ecosystem). It demonstrates Sourcey rendering the official `meilisearch-openapi.json` OpenAPI 3.1.0 specification into a multi-page HTML site with source-mapped API navigation, generated search, and Markdown guides.

The Meilisearch project is the target for this second-ecosystem proof. It is a maintained, third-party, MIT-licensed Rust HTTP search server. The OpenAPI spec used here is pinned to the official v1.49.0 release artifact (`meilisearch-openapi.json`, commit SHA `17758dce3fcc21d8e2415176c29d1106e905de00`) and includes 69 distinct paths with 139 operations.
