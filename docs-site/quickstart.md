# Quickstart

This site was built with [Sourcey 3.6.5](https://github.com/sourcey/sourcey) using the `openapi` and `markdown` adapters.

## Build locally

```bash
npm install
npx sourcey build
```

The output is a static `dist/` directory that you can serve from any web host.

## Pages

- **Getting Started** — overview, quickstart, concepts, adapter notes
- **API Reference** — every operation from the Meilisearch OpenAPI 3.1.0 spec, with request/response schemas and code samples
